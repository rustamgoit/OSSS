# Завдання 1. Огляд активних процесів

### 1. Вивести список усіх процесів

```bash
ps -ef
```

### 2. Запустити інтерактивний монітор

```bash
top
```

**Результат:**

Найбільше оперативної пам'яті використовував процес **systemd-journald** (`systemd-journal`).

### 3. Знайти PID поточної оболонки

```bash
echo $$
```

Результат:

```text
291
```

# Завдання 2. Робота у фоні та керування процесами

### 1. Запустити процес у фоновому режимі

```bash
sleep 1000 &
```

### 2. Переглянути список фонових завдань

```bash
jobs
```

### 3. Повернути процес на передній план

```bash
fg %1
```

### 4. Зупинити процес та завершити його

```bash
jobs -l
kill -9 22378
```

### 5. Запустити команду за допомогою `nohup`

```bash
nohup sleep 1000 &
```

# Завдання 3. Пріоритети та обмеження

### 1. Запустити команду з нижчим пріоритетом

```bash
nice -n 10 sleep 1000 &
```

### 2. Змінити пріоритет вже запущеного процесу

```bash
renice 15 -p 23048
```

### 3. Переглянути обмеження ресурсів

```bash
ulimit -a
```

# Завдання 4. Моніторинг ресурсів

### 1. Переглянути використання дискового простору

```bash
df -h
```

### 2. Переглянути використання оперативної пам'яті

```bash
free -h
```

Результат:

```text
Mem:   15 GiB total, 564 MiB used, 14 GiB free
Swap:   4 GiB total, 0 B used, 4 GiB free
```
